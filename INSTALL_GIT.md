# Install Git - Quick Guide

## Download & Install

1. **Go to:** https://git-scm.com/download/win
2. **Click:** "Download for Windows" (64-bit)
3. **Run** the downloaded installer
4. **Click "Next"** through all screens (defaults are fine)
5. **Click "Install"**
6. **Restart PowerShell** after installation

## Verify Installation

Open PowerShell and type:
```powershell
git --version
```

You should see: `git version 2.xx.x`

If you see an error, restart your computer and try again.
